## 	我们有几种方式可以隐藏一个元素：

CSS display的值是none。
	type="hidden"的表单元素。
	宽度和高度都显式设置为0。
	一个祖先元素是隐藏的，该元素是不会在页面上显示
	CSS visibility的值是hidden
	CSS opacity的指是0
	如果元素中占据文档中一定的空间,元素被认为是可见的。
	可见元素的宽度或高度，是大于零。
	元素的visibility: hidden 或 opacity: 0被认为是可见的，因为他们仍然占用空间布局。
	不在文档中的元素是被认为是不可见的，如果当他们被插入到文档中，jQuery没有办法知道他们是否是可见的，因为元素可见性依赖于适用的样式
