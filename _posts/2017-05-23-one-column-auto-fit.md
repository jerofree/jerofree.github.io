## 一列宽度自适应

```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <style>
        #layout{
            background-color: #cccccc;
            border:2px solid #333333;
            width: 80%;
            height: 300px;
        }
    </style>
</head>
<body>
<div id="layout">1列固定宽度</div>
</body>
</html>
```
